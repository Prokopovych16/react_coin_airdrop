# react_coin_airdrop

**To launch the project you should**

Clone the project
```
  git clone https://github.com/Prokopovych16/react_coin_airdrop.git
```

Switch to the folder with code

```
  cd react_coin_airdrop
  cd frontend
```

Install required dependencies

```
  npm i
```

Start the project

```
  npm start
```
